# 三国杀助手 - 快速开始

## 使用方法

### 方式1：交互模式（推荐）
双击运行 `start.bat`，进入交互式问答模式：

```
==================================
   三国杀问答助手 - 交互模式
==================================
输入问题进行查询，输入 'quit' 或 'exit' 退出

👤 你: 诸葛亮的技能是什么？
🤖 助手: 【诸葛亮】迟暮的丞相
势力：蜀 | 体力：3

技能：
  【观星】(准备阶段技)：准备阶段，你可以观看牌堆顶的X张牌...
  【空城】(锁定技)：若你没有手牌，你不能成为【杀】或【决斗】的目标。
```

### 方式2：代码调用
```python
from assistant import SanguoshaAssistant

assistant = SanguoshaAssistant()

# 查询武将
print(assistant.ask("赵云的技能是什么？"))

# 查询卡牌
print(assistant.ask("无中生有怎么用？"))

# 查询规则
print(assistant.ask("判定是什么？"))
```

## 功能特点

- 🎴 **武将查询**：10+ 经典武将（刘备、关羽、诸葛亮、曹操、孙权等）
- 🃏 **卡牌查询**：基本牌、锦囊牌、装备牌全覆盖
- 📜 **规则解答**：游戏规则、FAQ 解答
- 🔍 **智能匹配**：支持模糊查询和关键词匹配

## 数据文件

- `data/heroes.json` - 武将数据
- `data/cards.json` - 卡牌数据  
- `data/rules.json` - 规则数据

可以自行编辑这些 JSON 文件来添加更多内容！
